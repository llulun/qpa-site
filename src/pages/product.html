安装指南

  1. 安装扩展：Tampermonkey（推荐）或 Violentmonkey。
  2. 获取脚本：从官方仓库/Release 下载 QPA.js。
  3. 导入：扩展 Dashboard “导入文件”或创建新脚本粘贴内容。
  4. 域名匹配：*://*.qzone.qq.com/*。

使用指南

  • AL Menu：页面右上角浮动菜单，进入控制面板。
  • 状态栏：查看任务进度、剩余时间与提示信息。
  • 日志：面板查看、清除与导出。
  • 操作：暂停/恢复与测试执行。

配置参数字典

字段 类型 默认值 推荐值 说明
refreshIntervalNumber180180–300刷新频率（秒）
likeDelayNumber53–7点赞延迟（秒）
scrollStepNumber480360–600滚动步长（像素）

示例代码

// 调整延时与滚动策略
config.likeDelay = 5;
config.scrollStep = 480;
config.randomizeDelay = true;

// 启用日志
logger.setLevel('INFO');
logger.maxEntries = 500;

// 暂停与恢复
scheduler.pause();
setTimeout(() => scheduler.resume(), 30000);
